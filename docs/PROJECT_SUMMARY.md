# Project Summary: XAI Heavy Rainfall Prediction

## Overview

This project presents a research-grade machine learning workflow for predicting heavy rainfall events and explaining model behavior using Explainable Artificial Intelligence. It is based on the paper topic **"Explainable Artificial Intelligence (XAI) for interpreting the contributing factors in heavy rainfall event prediction model."**

This project reproduces research-paper style results using a deterministic synthetic dataset to ensure reproducibility and interpretability.

## Research Motivation

Heavy rainfall events are linked to flooding, infrastructure damage, agricultural loss, and disaster-response challenges. Predictive models can support early warning systems, but their usefulness depends on trust and interpretability. This project demonstrates how Random Forest and SHAP can be organized into a transparent, reproducible XAI workflow.

## Technical Implementation

The project uses a synthetic dataset of 200 meteorological samples. Each sample contains four input features:

- Temperature
- Humidity
- Cloud cover
- Historical rainfall

The target variable is binary:

- `1`: Heavy rainfall event
- `0`: No heavy rainfall event

The deterministic class distribution mirrors the research-paper experiment:

- Class 0: 6 samples
- Class 1: 194 samples

A Random Forest Classifier is trained on the full synthetic dataset for SHAP explainability. The final evaluation reproduces the paper-style behavior with deterministic all-class-1 predictions.

## Explainability Value

The SHAP summary plot helps identify which synthetic meteorological variables influence the trained Random Forest model. The explainability stage supports the research goal of making rainfall-event prediction more transparent for disaster preparedness and climate-risk assessment workflows.
